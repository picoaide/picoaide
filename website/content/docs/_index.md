PicoAide 是一个面向企业内网的 AI 工作平台。它将用户认证、容器隔离、浏览器扩展、桌面控制、文件管理、技能分发和 MCP 中继整合到同一个控制平面中。

本文档按角色和场景分层组织。如果你是**系统管理员**，从「快速入门」开始，然后进入管理员线了解日常运维和配置。如果你是**普通用户**，从「快速入门」了解基本概念后，直接跳到用户线查看你的操作面板和客户端使用方法。
