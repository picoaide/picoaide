PicoAide 是一个面向企业内网的 AI 工作平台。它将用户认证、容器隔离、浏览器扩展、桌面控制、文件管理、技能分发和 MCP 中继整合到同一个控制平面中。

本文档按真实使用顺序组织：

1. 从 [快速开始](/docs/quick-start/) 完成安装和首次初始化
2. 通过 [架构概览](/docs/architecture/) 理解服务边界和连接路径
3. 在 [配置参考](/docs/configuration/) 中查看配置结构和文件布局
4. 通过 [浏览器扩展](/docs/browser-extension/) 和 [桌面客户端](/docs/desktop-client/) 接入执行端
5. 使用 [技能系统](/docs/skills/) 和 [API 参考](/docs/api/) 做集成和自动化

如果你只想尽快验证部署，直接从快速开始进入。若你需要排查行为是否和代码一致，优先查架构和 API 页面。
