普通用户登录 PicoAide 后，进入 `/manage` 页面。你的个人管理面板包含六个功能标签页，涵盖 AI 助手的日常配置和使用。如果使用超管账号访问，会被自动重定向到管理后台。

## 登录面板

1. 在浏览器中访问 PicoAide 服务器地址（如 `http://picoaide.example.com`）
2. 页面自动跳转到登录页面
3. 输入你的用户名和密码
4. 点击「登录」
5. 登录成功后自动进入个人面板

如果登录失败，确认：
- 用户名和密码是否正确
- 账号是否在服务器上存在（LDAP 模式下需要先在 LDAP 中创建用户）
- 服务器管理员是否将你加入了白名单（如果白名单已启用）

## 面板概览

管理面板顶部显示你的用户名和角色信息。下方是四个功能标签页：

| 标签页 | 功能 |
|--------|------|
| 频道 | 查看和配置 AI 可用渠道 |
| 文件 | 管理工作区中的文件 |
| 技能 | 查看已安装技能，安装或卸载技能 |
| 团队空间 | 访问团队共享文件夹 |
| Cookie 管理 | 查看和撤销已授权的 Cookie 域名 |
| 密码 | 修改本地密码 |

## 频道管理

频道是 AI Agent 可以使用的工具和能力模块，例如网页搜索、图片生成、代码执行等。超管可以设置全局可用的频道列表，你可以在自己的面板中启用或禁用。

### 查看频道列表

在「频道」标签页中可以看到所有可用频道及其当前状态：

- **允许**：管理员允许此频道
- **已启用**：你已开启此频道（需要在管理员允许的前提下）
- **已配置**：此频道需要额外配置字段

### 启用/禁用频道

1. 找到你要设置的目标频道
2. 点击开关切换启用/禁用状态
3. 如果是首次配置，可能需要填写配置字段
4. 保存后，系统会自动重启你的 AI 容器使配置生效

### 配置渠道字段

某些频道需要额外配置才能使用，例如：

- 网页搜索频道可能需要配置搜索引擎 API key
- 图片生成频道可能需要配置图像模型参数
- 第三方集成频道可能需要填写访问凭证

在频道管理中找到需要配置的频道，点击配置按钮，填写所需字段后保存。

## 文件管理

你的工作区位于 `users/<你的用户名>/.picoclaw/workspace/`。这个目录是你在 AI 容器中的工作空间，AI Agent 可以在这里读写文件。

### 浏览文件

在「文件」标签页中可以看到工作区的内容结构：

- 根目录显示你所有的文件夹和文件
- 点击文件夹进入子目录
- 页面顶部显示当前路径（面包屑导航）

### 上传文件

1. 点击「上传文件」按钮
2. 从本机选择要上传的文件（最大 32MB）
3. 确认上传
4. 文件会出现在当前目录中

支持的用途：上传配置文件、数据文件、文档等供 AI 处理。

### 下载和删除

- **下载**：点击文件旁的下载按钮，将文件保存到本机
- **删除**：选中文件或目录，点击删除按钮（删除操作不可撤销）

### 创建目录和编辑文件

- **创建目录**：在当前目录下创建新的文件夹
- **在线编辑**：对于文本文件（如 `.txt`、`.json`、`.md`、`.yaml` 等），可以直接在浏览器中编辑保存。点击文件即可查看内容，点击编辑按钮修改后保存。

## 技能管理

在「技能」标签页中可以管理你的 AI 技能：

- **查看已安装技能**：列出当前已部署到工作区的所有技能
- **安装技能**：从可用技能列表中选择并安装
- **卸载技能**：移除不再需要的技能

相关 API：

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| `GET` | `/api/user/skills` | 查看已安装技能 |
| `POST` | `/api/user/skills/install` | 安装技能 |
| `POST` | `/api/user/skills/uninstall` | 卸载技能 |

## 团队空间

团队空间是管理员创建的共享文件夹，所有用户都可以访问。适合存放：

- 团队共用的配置文件模板
- 共享的数据文件
- 团队文档和知识库

团队空间中的文件对所有用户可见。你对团队空间中的文件有哪些操作权限取决于管理员的配置。

## Cookie 管理

在「Cookie 管理」标签页中可以查看和管理已授权的 Cookie 域名：

- **查看已授权域名**：列出已授权给 AI Agent 读取的 Cookie 域名
- **撤销授权**：删除域名授权，AI Agent 将无法访问该站点的 Cookie

相关 API：

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| `GET` | `/api/user/cookies` | 查看已授权 Cookie 域名 |
| `POST` | `/api/user/cookies/delete` | 取消 Cookie 域名授权 |

## 密码修改

在「密码」标签页可以修改你的登录密码：

**本地模式**：
1. 输入当前密码
2. 输入新密码
3. 再次输入新密码确认
4. 点击保存

**统一认证模式（LDAP/OIDC）**：
在统一认证模式下，密码由公司认证中心管理，不能在 PicoAide 中修改。请联系 IT 部门或使用公司密码修改系统。

## MCP Token

在 Web 面板中你可以获取自己的 MCP Token。Token 用于：

- AI 容器通过 MCP SSE 端点连接服务
- 浏览器扩展和桌面客户端 WebSocket 认证

Token 在首次请求时自动生成，格式为 `用户名:随机hex`。你可以复制后配置到容器的 `config.json` 中。

> MCP Token 是敏感信息，不要分享给他人。如果需要刷新 Token，联系管理员重置。
