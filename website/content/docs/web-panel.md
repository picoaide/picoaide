普通用户登录 PicoAide 后，进入 `/user/*` 页面。你的个人面板包含九个功能页面，涵盖 AI 助手的日常配置和使用。如果使用超管账号访问，会被自动重定向到管理后台。

## 登录与首页

1. 在浏览器中访问 PicoAide 服务器地址（如 `http://picoaide.example.com`）
2. 页面自动跳转到登录页面
3. 输入你的用户名和密码；如果是 SSO 模式，点击 SSO 登录按钮
4. 登录成功后自动进入个人面板

**首页**（`/user/welcome`）显示浏览器扩展和桌面客户端的下载和安装指引。系统会自动检测你的浏览器类型（Chrome/Edge/Firefox/Safari）和操作系统（Windows/Mac/Linux），展示对应的安装步骤。

如果登录失败，确认：
- 用户名和密码是否正确
- 账号是否在服务器上存在（LDAP 模式下需要先在 LDAP 中创建用户）
- 服务器管理员是否将你加入了白名单（如果白名单已启用）

## 面板导航

登录后顶部显示用户名和角色信息。下方是九个功能标签页：

| 标签页 | 功能 |
|--------|------|
| 首页 | 客户端安装指引和系统信息 |
| 对话 | AI 实时对话、消息流式输出 |
| 技能中心 | 查看已安装技能，安装/卸载技能 |
| 通讯渠道 | 配置 AI 可用渠道和参数 |
| 文件管理 | 管理工作区中的文件 |
| 团队空间 | 访问团队共享文件夹 |
| AI 授权 | 查看和撤销已授权的 Cookie 域名 |
| 修改密码 | 修改本地密码 |
| 定时任务 | 创建和管理 AI 定时任务 |

## 对话

「对话」页面是与 AI 实时交互的核心界面。

### 发送消息

1. 在底部的输入框中输入消息
2. 按回车或点击发送按钮
3. AI 通过 SSE 流式返回响应内容

### 流式输出

AI 的响应会实时流式显示在对话区，支持以下事件类型：

| 事件 | 说明 |
|------|------|
| `user_message` | 用户消息回显 |
| `text_delta` | AI 文本增量输出 |
| `tool_call_start` | AI 开始调用工具 |
| `tool_result` | 工具调用结果 |
| `progress` | 处理进度信息 |
| `finish` | 对话完成 |

### Markdown 渲染

AI 的响应内容会渲染为 Markdown 格式，支持代码高亮、表格和链接。工具栏调用信息会以折叠面板形式展示。

### 断线重连

如果页面意外刷新或网络断开，系统会自动尝试恢复之前的对话会话。对话历史存储在 `runId` 中，支持页面可见性变化时的自动重连。

### 停止对话

点击停止按钮可以中断 AI 正在生成的响应。

## 通讯渠道

渠道是 AI Agent 可以使用的工具和能力模块，例如网页搜索、图片生成、代码执行等。超管可以设置全局可用渠道，你可以在此页面中启用或配置自己的渠道。

### 查看渠道列表

左侧列表显示所有可用渠道及其状态：

- 绿色标签：已启用
- 灰色标签：已禁用
- 图标状态指示是否已配置

### 配置渠道参数

点击渠道进入配置页面。不同类型的字段使用不同的控件：

| 字段类型 | 控件 | 说明 |
|---------|------|------|
| 文本 | 输入框 | 普通文本参数 |
| 密码 | 密码输入框 | API Key 等敏感信息 |
| 布尔值 | 开关控件 | 启用/禁用选项 |
| 数字 | 数字输入框 | 数值参数 |
| 数组 | 多行文本框 | 每行一个元素 |
| JSON/对象 | 多行文本框 | 格式化的 JSON 内容 |

保存后系统会自动更新 IM 连接，无需手动重启。

## 文件管理

你的工作区位于沙箱内的 `/root` 目录（宿主机上对应 `users/<你的用户名>/`）。这是 AI Agent 可以读写的工作空间。

### 浏览文件

文件管理页面显示工作区的内容结构：

- 根目录显示所有文件夹和文件
- 点击文件夹进入子目录
- 页面顶部显示当前路径（面包屑导航）
- 文件带有类型标签，鼠标悬停显示说明

### 操作文件

| 操作 | 说明 |
|------|------|
| 上传文件 | 选择本地文件上传（最大 32MB） |
| 下载文件 | 点击下载按钮保存到本地 |
| 删除文件 | 删除文件或目录（不可撤销） |
| 创建目录 | 在当前目录下新建文件夹 |
| 编辑文件 | 在线编辑文本文件并保存 |

### 文件标签说明

工作区中的文件带有语义标签，帮助理解目录用途：

| 标签 | 说明 |
|------|------|
| skills | 已部署的技能目录（只读） |
| memory | AI 长期记忆存储 |
| session | 对话历史记录 |
| archive | 归档文件 |

技能目录中的文件为只读，不允许上传、编辑或删除。

## 技能中心

在「技能中心」页面可以管理你的 AI 技能：

- **查看技能**：以卡片形式展示所有技能，包括已安装、组部署和可用三种状态
- **搜索技能**：输入关键词筛选
- **安装技能**：从可用技能列表中安装（需超管开放权限）
- **卸载技能**：移除已安装的技能

## 团队空间

团队空间是管理员创建的共享文件夹。你可以在该页面查看所有可访问的共享文件夹，包括：

- 文件夹名称和图标
- 公开/私有标识
- 描述信息和成员数量
- 点击查看成员列表

团队空间中的文件在 AI 沙箱中以只读方式挂载。

## AI 授权

Cookie 授权管理页面显示已授权给 AI Agent 读取的 Cookie 域名列表。每个条目显示：

- 域名
- 授权时间
- 撤销按钮

AI Agent 需要在 `.security.yml` 中配置 Cookie 才能访问特定网站。当你通过浏览器扩展同步 Cookie 后，对应域名会出现在此列表中。

## 定时任务

「定时任务」页面可以创建和管理 AI 定时任务。

### 创建任务

1. 点击「新建任务」按钮
2. 输入 Cron 表达式（如 `0 9 * * 1-5` 表示工作日早 9 点）
3. 输入任务提示词（AI 执行任务的指令）
4. 选择发送渠道
5. 保存创建

### 管理任务

| 操作 | 说明 |
|------|------|
| 启用/禁用 | 切换任务运行状态 |
| 编辑 | 修改任务的表达式、提示词和渠道 |
| 删除 | 移除定时任务 |

系统会显示每个任务的状态、最近运行时间和下次运行时间。

### Cron 表达式格式

支持标准 5 段 Cron 表达式：
- `* * * * *` — 每分钟（分钟 小时 日 月 周）
- `0 9 * * 1-5` — 工作日上午 9 点
- `*/30 * * * *` — 每 30 分钟

## 修改密码

在「修改密码」页面可以修改你的登录密码。

**本地模式**：输入当前密码和新密码（至少 6 位），确认后保存。

**统一认证模式（LDAP/OIDC）**：密码由公司认证中心管理，不能在 PicoAide 中修改。请联系 IT 部门或使用公司密码修改系统。
