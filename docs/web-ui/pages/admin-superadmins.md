# 超管账户页面

## 页面路由

`/admin/superadmins`

## 权限要求

超管（`superadmin`）登录

## 功能概述

管理系统超级管理员账户，包括查看超管列表、创建新超管、删除超管、重置超管密码。

## 功能详细说明

### 1. 超管列表

**功能**：展示所有超管账户

**API 端点**：`GET /api/admin/superadmins`

**响应格式**：
```json
{
  "success": true,
  "admins": ["admin", "admin2"]
}
```

### 2. 创建超管

**触发方式**：点击"创建超管"按钮 → 输入用户名 → 系统自动生成密码 → 展示密码

**API 端点**：`POST /api/admin/superadmins/create`

**请求格式**（form）：
```
username=newadmin&csrf_token=xxx
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| username | string | 是 | 超管用户名 |

**响应格式（成功）**：
```json
{
  "success": true,
  "message": "超管创建成功",
  "username": "newadmin",
  "password": "autoGeneratedSecurePassword123!"
}
```

**密码仅在创建时返回一次**，前端必须展示给用户，并提示用户保存。

**响应格式（失败）**：
```json
{
  "success": false,
  "error": "用户名已存在"
}
```

### 3. 删除超管

**触发方式**：点击超管行的"删除"按钮 → 确认删除

**API 端点**：`POST /api/admin/superadmins/delete`

**请求格式**（form）：
```
username=oldadmin&csrf_token=xxx
```

**响应格式（成功）**：
```json
{
  "success": true,
  "message": "超管 oldadmin 已删除"
}
```

**响应格式（失败）**：
```json
{
  "success": false,
  "error": "至少保留一个超管账户"
}
```

**限制**：必须至少保留一个超管账户。如果系统只有一个超管，删除操作会被拒绝。

### 4. 重置超管密码

**触发方式**：点击超管行的"重置密码"按钮 → 确认重置

**API 端点**：`POST /api/admin/superadmins/reset`

**请求格式**（form）：
```
username=admin&csrf_token=xxx
```

**响应格式（成功）**：
```json
{
  "success": true,
  "message": "密码已重置",
  "password": "newGeneratedSecurePassword456!"
}
```

**新密码仅在重置时返回一次**，前端必须展示给用户。

## 涉及的 API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/admin/superadmins` | GET | 超管列表 |
| `/api/admin/superadmins/create` | POST | 创建超管 |
| `/api/admin/superadmins/delete` | POST | 删除超管 |
| `/api/admin/superadmins/reset` | POST | 重置超管密码 |
